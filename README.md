##### 任务内容：

1. 根据所给的raw_info文件夹，编写一个python程序，功能是生成2个csv表，分别记录所有出现过的队伍的信息，以及所有出现过的成员（学生，老师）的个人信息。
2. 队伍信息包含的字段：队伍名称，队长姓名，队长电话。
3. 成员个人信息包含的字段：姓名，身份（队长/队员/指导老师），电话，学号（如果有的话）。



##### 仓库结构：

- raw_info-原始数据文件夹
- output-输出文件夹
- src-代码文件夹
  - conf储存常用变量
  - main为程序入口



##### 完成情况：
